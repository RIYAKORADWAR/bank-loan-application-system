Bank Loan Application System (BLAS)

A digital Bank Loan Application System (BLAS) project that automates the complete loan lifecycle from application submission to loan disbursement. This project demonstrates Business Analysis, workflow design, and banking process modeling concepts used in real-world fintech and banking environments.

---

## Project Overview

The system models the complete journey of a loan application:

Customer Application → Document Upload → KYC Verification → Credit Assessment → Approval Workflow → Sanction Generation → Pre-Disbursement → Loan Disbursement

This repository includes detailed business analysis and process design deliverables commonly used in enterprise banking projects.

## Key Features

* Multi-step digital loan application workflow
* OCR-based document verification
* Aadhaar eKYC & Video KYC (V-CIP)
* Automated credit bureau integration
* Credit scorecard and underwriting workflow
* Multi-level approval system
* Maker-checker authorization controls
* Digital sanction letter generation
* CBS (Core Banking System) integration
* NEFT/RTGS loan disbursement
* NACH eMandate registration
* SLA tracking and escalation workflow
